O metodo toString() converte um numero em string

O metodo toExponential() retorna uma string, com o numero arredondado e escrito usando a notação exponencial. O parametro opcional define a quentidade de digitos apos a casa deceimal.

O metodo toFixed() arredonda o numero para o dado numero de digitos. Para trabalhar com dinheiro o ideal é usar o toFixed(2);

O metodo toPrecision() retorna uma string, com um numero escrito com o tamanho especificado
